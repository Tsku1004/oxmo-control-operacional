# Cambios v20260626 - Dashboard Gerencial V2

Se actualizó el perfil **Gerente** con un dashboard ejecutivo de estilo industrial/gerencial:

- KPIs principales con tarjetas y mini tendencias.
- Gráfico grande de tendencia operacional (producción / consumo / ritmo inventario).
- Distribución por clasificación y estado de lotes.
- Bloques de masa por sector.
- Indicador de cumplimiento de especificación.
- Tarjeta de fino Mo acumulado.
- Tarjetas de nivel de silos.
- Tabla ejecutiva por sector.
- Panel de alertas y observaciones.
- Indicador visual de estado de nube en perfil Gerente.
- `index.html` con nueva versión de caché para forzar actualización.

## Archivos clave
- `app.js`
- `styles.css`
- `index.html`

Subir estos archivos a la raíz del repositorio y luego abrir la app con `Ctrl + F5`.
